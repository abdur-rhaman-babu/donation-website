What is document object model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, where each part of the document (such as elements, attributes, and text content) is modeled as an object.

How do you select an element from the DOM?

To select an element from the Document Object Model (DOM), you can use several methods provided by the DOM API. These methods allow you to target elements by various criteria like id, class, tag name, or more complex selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM where you leverage the concept of event bubbling to manage events more efficiently. Instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element.
